## Installation
Github:

    git clone https://github.com/I2rys/JVSTL
    
Python Packages:

    pip install -r requirements.txt

## Usage

    python index.py <javascript_link>
    
+ javascript_link - The link of the javascript to scan for vulnerabilities.
